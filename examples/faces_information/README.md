# Faces Information Example

Demonstrates how to query face data from a Voronoi diagram: cell faces, face neighbors, face vertices, and center of mass.

This is a **serial** example (no MPI, no VTK).

## Building

```bash
MADVORO_PREFIX=/path/to/madvoro/install BOOST_DIR=/path/to/boost bash build.sh
```

## Running

```bash
./test
```
